# 重复匹配的基本使用

```js
    /hy+/                           // y必须有多个
    /hy*/                           // y有0个或多个
    /hy{2}/                         // y有2个
    /hy{2,10}/                      // y有2个到10个
    /hy{2,}/                        // y有2个或2个以上以上
    /hy?/                           // y有0个或1个
```

# 重复匹配与原子组

```js
    /(hd)+/                         // 匹配多个 hd
```
