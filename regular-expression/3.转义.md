# 正则表达式的转义
 ```js
    /\d+/       // 匹配多个数字
    /\d+./      // .的意思是匹配除换行符以外的任何字符

    // 需要注意的是，用构造的方式创建正则和用字面量的形式创建正则是有略微的差别的，如：
    let reg = new RegExp('\d+\.\d+');
    let reg2 = /\d+\.\d+/;
    // 上面两个是有区别的，在用构造的方式创建正则的时候，是传入一个字符串作为参数的，这个时候 \. 就相当于 .
    // 解决方式是在每个斜杠之前再加一个斜杠，如下：
    let reg = new RegExp('\\d+\\.\\d+');
 ```

```js
   /https?\/\/\w+\.\w+\.com/      // ?表示有或者没有都行
                                  // http后面的两个 / 也是需要转义的
```
