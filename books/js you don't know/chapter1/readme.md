# 什么是作用域
 - 变量  程序的状态
 - 赋值过程   var a = 2;
    1. 分词/词法分析阶段
        [var,a,=,2]
        token  分析语法是否正确
    2. 语法分析阶段
        语法树🌲
        编译器也是一段需要运行的代码
        DOM树
        AST   Abstract Syntax Tree  抽象语法树
    3. 二进制代码生成

# LHS和RHS
    LHS：变量赋值或写入内存
    RHS：变量查找或从内存中读取
    严格模式下，不使用var关键字就直接创建一个变量并赋值的话，就会报错，非严格模式的话就会创建一个全局变量
